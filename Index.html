O que é o HTML!?

O HTMLé uma linguagem de marcação de hipertexto, e não é uma linguagem de programação. Foi desenvolvido por Tim Berns Lee. Tem o objetivo de desenvolver arquivos.html, e que são interpretados pelo navegador, gerando assim uma página. Um conjunto de mais de uma página pode ser chamado de site.
